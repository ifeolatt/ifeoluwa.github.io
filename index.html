Project Portfolio

Employee Attrition Analysis

Analysis was done to deduce key insights that affects employement attrition rate in an organization. The essence of measuring and analysing the attrition rate in an organization is to understand the key factors causing the loss of staff and develop effective staff retention strategies. Attrition rate helps an organization understand how well it is retaining its talents.

NYC Taxi Trip Analysis

Using data set that was collected over a period of four years (2017 - 2020). This data set was cleaned using PowerBi Power Query. With the clean dataset, I built a dashboard with visualizes key insights. Insights deduced from this project enables one spot trends of taxi trips in New York City over the period of the four year.

Airline and Flight Delay Analysis

A total approximately six million flight records from 2015 were cleaned and analysed which I visualized insight such as least popular and most popular airline, most preferred travel day of the week, number of cancelled flghts and many more.

Wine Tasting Analysis

For this project, analysis was carried out on wine tasters for various wineries and vineyard for which I generated visual representations that are easily assimilated.

Global Shark Attack Dashboard

I used a dataset that details reported shark attacks globally, over the period or 120 years. After cleaning this data using power query, created charts that showed the trend of shark attacks over 120 years, the most injured body parts, the time of the day during which one is likely to get attacked, the countries that reported the most attacks.etc.

Restaurant Rating Analysis

This is a detailed analysis of the ratings in a restaurat, the factors that influenced the consumer rating. etc.

Global Covid 19 Dashboard

I created a dashboard that shows the rising / falling numbers of covid 19 cases, affected countries and death rate globally.
